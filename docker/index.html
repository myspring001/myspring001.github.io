docker

2021-04-21后端 • docker
Docker 是一个开源的应用容器引擎
容器是完全使用沙箱机制，相互之间不会有任何接口,更重要的是容器性能开销极低。

获取镜像：
docker pull mysql 
启动容器：
docker run -it mysql /bin/bash
-i 交互式
-t 终端

查看镜像list:
docker ps -a

启动容器：
docker start docker容器id

停止容器：
docker stop docker容器id

exec 命令
docker exec -it 243c32535da7 /bin/bash
退出容器不会导致容器停止

删除容器：
docker rm -f docker容器id

随机使用端口
-P:将容器内部使用的网络端口随机映射到我们使用的主机上。
